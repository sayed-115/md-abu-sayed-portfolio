# Dynamic Portfolio + CV Generator

A modern, responsive personal portfolio built with Vanilla HTML, CSS, and JavaScript. Features a live dynamic GitHub projects feed, an integrated contact form, and one-click PDF CV export.

[![Live Demo](https://img.shields.io/badge/Live-Demo-0ea5e9?style=for-the-badge)](https://sayed-115.github.io/md-abu-sayed-portfolio/)
[![Vanilla Stack](https://img.shields.io/badge/Stack-HTML%20%7C%20CSS%20%7C%20JavaScript-111827?style=for-the-badge)](#tech-stack)

## 🚀 Live Demo

[View Live Portfolio](https://sayed-115.github.io/md-abu-sayed-portfolio/)

## ✨ Key Features

- **Modern Interactive UI**: Features a custom lightweight interactive "Dot Matrix" canvas background with parallax scroll effects.
- **Instant CV Export**: Generates a clean, interview-ready PDF directly from the browser using `html2pdf.js`.
- **Dynamic GitHub Feed**: Automatically fetches and displays recent active repositories from the GitHub API.
- **Serverless Contact Form**: Fully functional contact form powered by EmailJS.
- **No Frameworks**: Built entirely with Vanilla JS and Custom CSS for maximum performance and a deep understanding of core web technologies.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript (ES6+)
- **Integrations**: 
  - GitHub API (Dynamic Projects)
  - EmailJS (Contact Form)
  - `html2pdf.js` (Client-side PDF Generation)
  - Google Fonts & Font Awesome

## 💻 Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/sayed-115/md-abu-sayed-portfolio.git
   cd md-abu-sayed-portfolio
   ```
2. Open `index.html` in your browser or run using a Live Server extension in VS Code.
